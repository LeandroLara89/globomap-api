# List Collections

----
  Returns json data with list of collections in DB.

* **URL**

  /v1/collections

* **Method:**

  `GET`
  
* **Success Response:**

  * **Code:** 200
  * **Content:** `["coll1","coll2","coll3"]`

* **Sample Call:**

  ```shell
     curl http://localhost:5000/v1/collections
  ```
